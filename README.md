## MUNDIALPOO

## Descripción

Este proyecto fue desarrollado en Java utilizando Programación Orientada a Objetos. El sistema permite registrar selecciones, jugadores, directores técnicos, grupos y partidos, además de registrar resultados y mostrar la información del torneo.

## Funcionalidades

- Registrar selecciones.
- Registrar jugadores.
- Registrar directores técnicos.
- Crear grupos.
- Agregar selecciones a los grupos.
- Programar partidos.
- Registrar resultados de los partidos.
- Mostrar grupos y partidos.
- Guardar la información en un archivo.

## Clases principales

- Persona
- Jugador
- DirectorTecnico
- Seleccion
- Grupo
- Partido
- Torneo
- GestorArchivos
